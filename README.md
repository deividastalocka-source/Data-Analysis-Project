# Data Analysis Linear Regression

University project that involves analysing a real-world dataset using Microsoft Excel.



Project Files



Project 6 TV Marketing Regression.xlsx — The main Excel file containing the raw data, cleaned data, and analysis.

Project Report.pdf — A written report summarising the methodology, key findings, charts, and insights.

Project Presentation.pptx — A presentation highlighting the results and visuals from the project.



What Was Done



Performed calculations using Excel formulas and functions

Built visualizations to highlight trends and patterns

Summarized insights in both a report and a presentation



Tools Used



Microsoft Excel

Excel formulas (SUMIF, IF, LOOKUP functions, etc.)

Data cleaning and formatting tools



PDF report compilation

PowerPoint for presentation



Purpose of the Project



This project demonstrates practical data analysis skills using Excel, including data cleaning, visualization, and communication of insights through written and visual formats. It is intended for learning, practice, and portfolio demonstration.
